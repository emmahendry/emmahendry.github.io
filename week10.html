Task 3: What about other forms of accessibility?

The first way in which I would consider the topic of accessibility is through colour choice, by ensuring the colour pallet of a website-based project is understandable for users is significant. This can be accomplished through the understanding of the tools we used in week 10. Such as WebAIM, the contrast checker this website runs is a advantageous technique to get insight on how a websites design effects users ability to process the information.

Secondly, using fonts and language which remain legible for the majority of users can be beneficial for readability. This ties in with Readability Test tool we looked at this week. This allows developers to discover an average suitability based on the factor of age. Ensuring that a website remains educational and articulate, whilst maintaining a comprehensible reading style is remarkably important to a successful website design.

Font choice ties in with this aspect, as intricate or eccentric font choices can cause accessibility complications for users. When creating large portions of text for readers its key to ensure this is done in a manner which remains legible as fonts which take the practice of cursive writing can prove difficult to understand.
